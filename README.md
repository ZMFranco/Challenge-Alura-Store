Análisis de Facturación – Tiendas
Este proyecto tiene como objetivo analizar el desempeño de cuatro tiendas a partir de datos de ventas. Se evaluaron diferentes indicadores para determinar en qué tienda debería vender el Sr. Juan.

El análisis fue realizado en Google Colab utilizando Python y pandas.

Datos Utilizados

Se analizaron 4 archivos CSV correspondientes a:

Tienda 1
Tienda 2
Tienda 3
Tienda 4

Cada dataset contiene información sobre: Precio,Categoría del producto, Producto, Calificación y Costo de envío

Análisis Realizados

1️. Ingresos Totales

Se sumó la columna Precio para calcular la facturación total por tienda.

2️. Productos Vendidos por Categoría

Se agruparon los datos por categoría para identificar las más populares.

3️. Calificación Promedio

Se calculó el promedio de la columna Calificación para medir satisfacción del cliente.

4️. Productos Más y Menos Vendidos

Se contó la cantidad de ventas por producto para identificar los de mayor y menor rotación.

5️. Costo de Envío Promedio

Se calculó el promedio del Costo de envío por tienda.

Conclusión

Después de analizar ingresos, satisfacción del cliente y productos vendidos:

Con base en el análisis integral, se recomienda vender en la tienda 2 que presenta el mejor equilibrio entre ingresos altos y buena percepción del cliente.
